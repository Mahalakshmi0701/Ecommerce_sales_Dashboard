# 📊 Product Analytics: Data Cleaning, Transformation, and SQL Insights

## 📌 Project Overview

This project focuses on analyzing product data by performing data cleaning, transformation, and SQL-based analysis. The goal is to convert raw data into meaningful business insights.

---

## 🛠️ Tools & Technologies

* Python (Pandas)
* SQL (MySQL Workbench)
* Google Colab

---

## 🔄 Project Workflow

### 1. Data Cleaning (Python)

* Removed duplicate records
* Handled missing values
* Converted data types
* Filtered invalid data

### 2. Data Transformation

* Created new columns:

  * PriceCategory
  * RatingLabel
  * StockStatus

### 3. SQL Analysis

* Category-wise performance analysis
* Top-rated products identification
* Low stock detection
* Price vs Rating insights
* Ranking using window functions

---

## 📊 Key Insights

* Identified high-performing product categories
* Detected low stock items for inventory planning
* Found best value products based on price and rating
* Compared product performance within categories

---

## 📂 Project Files

* `products.csv` → Raw dataset
* `cleaned_products.csv` → Processed dataset
* `analysis.ipynb` → Python data cleaning & transformation
* `product_analysis.sql` → SQL queries

---

## 🎯 Conclusion

This project demonstrates end-to-end data analysis, from raw data processing to generating actionable insights using Python and SQL.
